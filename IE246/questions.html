  3. Write a Java code that prints all the letters in the alphabet in lowercase (move to the next line if e or t is to be printed next) with one space in between.
  4. We can cast a long into an int without any information loss. True or False?
  5. Write a Java code that prints the prime numbers between 350 and 2570.
