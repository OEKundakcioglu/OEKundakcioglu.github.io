Week 2

Hello World

  • IDEs are not particularly helpful while getting familiar with the syntax. Instead you are recommended to use a simple text editor.
  • For comments: Use // for single line, /* text... */ for multiple lines. It is important to document the flow of code with comments.
  • Java syntax for the class with the main method:
    class -classname- {
     public static void main (String[] args) {
      //code block here
     }
    }
  • Class name and file name must be same and extension must be .java
  • javac compiles the java file: "javac -classname-.java"
  • javac produces executable "-classname-.class"

Numerical Operations

  • We can define variables in 2 different ways:
    Variabletype -variablename- ;
    -variablename- = somevalue;
    // or alternatively
    Variabletype -variablename- = somevalue ;
  • Concatenation
    System.out.println("string " + variable) // outputs string variable
    System.out.println("string " + variable1 + variable2) // outputs string variable1variable2
    System.out.println("string " + (variable1 + variable2)) // outputs string (sum of variable1 and variable2)
  • Primitive Data Types:

    Source: book, page 33
  • integer / integer = integer. How to fix it, if a double result is needed?
    • Define integers as double.
    • Or convert to double: (double) var1/var2.
  • Do not rely on equality of doubles. When you manipulate doubles, although you do operations that lead to the same double mathematically, it is very likely that the results are not the same because of computer precision.
